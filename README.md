## WALLMART-FRONT

Requerimientos

- Node 8
- Docker 

#Para levantar localmente el proyecto ejecutar

 ```sh
 $ npm install
 ```

 ```sh
 $ npm start
 ```
Se levanta en [http://localhost:3000](http://localhost:3000)

#Para levantar el docker

```sh
 $ make create-image
 ```

 ```sh
 $ make run-local-image
 ```
